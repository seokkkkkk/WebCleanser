# netpuri 백엔드
- 텍스트 필터를 위한 flask, 사이트 링크 검사를 위한 spring으로 구성되어있습니다.
- 텍스트 필터 모델 학습 관련 정보는 netpuri_model에서 확인할 수 있습니다.
- 사이트 링크 검사를 위한 spring은 google safe browsing api를 사용하였습니다.
